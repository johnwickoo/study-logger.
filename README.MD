# Study Logger

A CLI tool to log daily study sessions and track weekly progress.

## What it does
- Logs study tasks, hours, problems solved, session rating and blockers
- Validates rating input (1-5 only)
- Generates a weekly summary with totals and average rating

## How to use
python3 exercise1.py   # log a session
python3 summary.py     # view weekly summary

## Built with
Python 3 — file I/O, loops, lists, input validation
